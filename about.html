Thank you for visiting.

Origami Note is a simple web app that will encode/decode your messages according to ROT13. It encodes letters: 'A' ↔ 'N', 'B' ↔ 'O' (and vice versa). Similarly, numbers will change: '1' ↔ '6', '2' ↔ '7', etc. If you liked this project or want to see more, check out my Github.
